Puppet Plugins

This is a Puppet Community curated list of plugins for Puppet and related tooling like puppet-lint and rspec-puppet. If you have a plugin you'd like to contribute, please submit a pull request!

Puppet Lint Plugins

Plugin Link Description
trailing_newline Extends puppet-lint to ensure that your manifest files end with newlines.
variable_contains_upcase Extends puppet-lint to ensure that your variables are all lower case.
param_docs A check for puppet-lint that validates all parameters are documented.
roles_and_profiles A puppet-lint plugin to check that a node definition declares only a role, a role class does not have any param and only declares profiles, and a profiles class can declare anything but a role.
vim_modeline A puppet-lint plugin to check for vim comment (modeline) as the last line in a manifest.
strict_indent Extends puppet-lint to ensure that your manifests follow a strict indentation pattern.
unquoted_string A puppet-lint plugin to check that selectors and case statements cases are quoted.
empty_string A puppet-lint plugin to check for variables assigned to the empty string.
spaceship_operator_without_tag A puppet-lint plugin to check that spaceship operator is called with a tag.
absolute_classname Puppet-lint plugin to check relative class name inclusions.
leading_zero A puppet-lint plugin to check for unquoted numbers with leading zero.
newmericvariable Extends puppet-lint to ensure that your variables are not numeric.
file_ensure A puppet-lint plugin to check the ensure attribute on file resources.
trailing_comma A puppet-lint plugin to check for missing trailing commas.
version_comparison A puppet-lint plugin to check for versions compared as numbers.
global_resource Extends puppet-lint to ensure that your manifests have no global resources.
appends A puppet-lint plugin to check that the appends operator (+=) is not used (removed in Puppet 4.0.0).
classes_and_types_beginning_with_digits A puppet-lint plugin to check for types and class names that begin with digits.
file_source_rights A puppet-lint plugin to check file rights when providing a source.
alias A puppet-lint plugin to check for alias parameters in resources.
security Checks puppet manifests for security related problems.
usascii_format A puppet-lint plugin to check that manifest files contain only US ASCII.
package_ensure A puppet-lint plugin to check the ensure attribute on package resources.
resource_reference_syntax Extends puppet-lint to ensure that the reference syntax follows Puppet 4 style.
undef_in_function A puppet-lint plugin to check for undef in function calls.
fileserver A puppet-lint plugin to check if puppet:/// is used instead of file().
